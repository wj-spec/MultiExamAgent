# IntelliExam-Agent 开发现状报告 (2026-03-27)

## 一、 产品定位
IntelliExam-Agent 是一个 **AI Native 的智能命题与审题工作台**。它从传统的“对话驱动”工具升级为了基于多智能体协作（Multi-Agent）的专业级教育辅助平台，核心理念是“基于自然语言发起任务，AI自动规划并分步执行，人类介入反馈迭代”。

## 二、 主要功能
1. **三场景独立工作区**：提供对话（Chat）、命卷（Proposition）和审卷（Review）三大专属核心场景，支持智能体在意图识别后自动引导跨场景切换。
2. **可视化待办看板（Todo Board）**：提供类似自动化流水线的卡片式任务看板，实时展示任务规划、执行进度、依赖关系及最终结果。
3. **专家级教育逻辑**：系统内部 Prompt 深度集成了布鲁姆教育目标分层、双向细目表、难度系数控制等专业命题规范。
4. **双流审题批注视图**：支持左侧原稿视图与右侧审查时间线的对照联动，提供高亮、定位和详尽的审题报告。
5. **动态思考链（Thought Chain）展示**：前端精细化渲染 Agent 的内部思考过程、工具调用日志和验证步骤，操作完全透明。
6. **记忆与自我进化机制**：
   - **短期记忆**：维持当前工作流与多轮对话的上下文。
   - **长期记忆**：系统自动沉淀用户的优质命题经验、参数规范和特殊偏好至本地 JSON 库，建立跨视窗的长效记忆检索体系。

## 三、 产品逻辑 (工作流机制)
系统采用极为灵活的 **Planner-and-Solver（规划与执行）** 模式，实现高度的自动化与人机协作闭环：
1. **意图路由（Router）**：`Chat Agent` 接收用户自然语言需求，精准判断用户意图，若为复杂命题或审题则携带上下文无缝切换至相应专业场景。
2. **任务规划（Planner）**：`Planner Agent` 根据历史记忆和当前目标，生成高度结构化的“待办任务清单（Todo List）”。例如：出卷任务会被拆分为核心考点分析、选择题生成、解答题生成、全卷质量核查等一系列节点。
3. **人类确认（Human-in-the-loop）**：用户可以对系统给出的大纲和待办事项进行确认，或是通过“新增讨论”指导修正方案重新规划（Re-plan）。
4. **任务调度与执行（Solver）**：`Solver Agent` 按照任务依赖树自动按序或并行执行。执行过程中，Agent 动态挂载 RAG（检索增强生成）、外部搜维、LLM能力等工具链。
5. **反思自检闭环（Reflect/Audit）**：Solver 内部针对每一道生成的试题触发“科学性、规范性”等质量审核机制，遇到不合格项触发反思机制重构原题（支持最高3轮重试），确保严谨的输出结果。

## 四、 功能实现与技术栈现状 (v3.1)
项目目前处于高速迭代后的 **3.1.0** 版本，整体核心功能实现已达 97% 左右。
* **底层系统架构**：服务端依靠 `Python 3.10+` 和 `FastAPI` 构建，多智能体流转核心依托 `LangGraph` 驱动其状态机（引入了全新的 AgentStateV3 结构）。
* **数据存储与持久化**：采用 `SQLite` 高效管理任务组（todo_groups）、任务流（todo_tasks）和用户互动评论；利用 `JSON` 文件轻量化持久存储长期经验长效记忆；搭配 `Chroma` 本地向量数据库提供业务相关知识库检索。
* **通信协议设计**：全盘依托全双工 `WebSocket` 协议取代轮询长连接，实现如 `todo_task_update`、`status_update`、`content_delta` 流式传输以及 Agent `interrupt_request` 等复杂状态的低延迟双向推送。
* **前端应用层**：秉持高性能、无绑定的“极客精神”，摒弃厚重前端框架，使用纯 HTML/CSS/Vanilla JS 结合 `marked.js` 重新构建现代 UI。实现了兼顾深浅色主题适配的毛玻璃流光看板、逐题扫描动效、进度矩阵骨骼屏和代码层级高亮展示等复杂交互。

## 五、 待完成与后续规划
虽然核心机制均已贯通，以下模块仍在规划列队中：
1. **多模态与文档解析深化**：优化审题模块针对 PDF 和 Word 的富文本甚至公式的解析能力。
2. **MCP 与动态技能扩展**：赋予 Chat/Solver Agent 支持基于 Model Context Protocol (MCP) 动态加载系统或组织内部工具箱的能力。
3. **成果导出机制**：建立完整的渲染模板，实现真题定稿或审卷报告一键生成并导出为可编辑的标准化 Word、PDF 文档。
